# 📝 TextUtils – Django Text Processing Web App

## 📌 Overview
TextUtils is a simple and efficient web application built using Django that allows users to perform various text operations quickly. It helps in cleaning and transforming text with just a few clicks.

---

## 🚀 Features
- 🔠 Convert text to UPPERCASE
- 🔡 Convert text to lowercase
- 🔄 Remove extra spaces
- ❌ Remove punctuation
- 📊 Word counter

---

## 🛠️ Tech Stack
- Python 🐍
- Django 🌐
- HTML, CSS, Bootstrap 🎨
